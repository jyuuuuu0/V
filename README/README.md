# V (브이)

팀 데일리 스크럼 자동화 및 PR 리뷰 리마인드 Discord 봇

## 기능

### 자동 스케줄러

- 오전 8시 데일리 스크럼 스레드 자동 생성
- 전날 미완료 항목 다음날 스크럼에 자동 표시
- 오후 12시 40분 미작성자 리마인드
- 오후 11시 마감 알림
- 매 30분마다 2시간 이상 리뷰 없는 PR @FE 멘션 알림

### 명령어

| 명령어        | 설명                      |
| ------------- | ------------------------- |
| `!todo`       | 오늘 팀원 전체 할 일 조회 |
| `!prlist`     | 리뷰 대기 PR 목록 조회    |
| `!scrumstats` | 내 스크럼 완료율 통계     |
| `!도움말`     | 명령어 목록 안내          |

### 스크럼 작성 방법

스레드 안에 아래 형식으로 작성

```
1. 어제 한 일: 내용
2. 오늘 할 일: 내용
3. 이슈 / 도움 필요한 점: 내용
```

작성 후 완료하면 ⭕, 못 마치면 ❌ 리액션

> ❌ 리액션을 누르면 다음날 스크럼 스레드에 미완료 항목이 자동으로 표시됩니다.

## 기술 스택

- Python
- discord.py
- aiosqlite (SQLite)
- Flask (GitHub webhook)
- Railway (배포)

## 환경변수

| 변수명            | 설명             |
| ----------------- | ---------------- |
| `DISCORD_TOKEN`   | Discord 봇 토큰  |
| `CHANNEL_ID`      | 포럼 채널 ID     |
| `GUILD_ID`        | 서버 ID          |
| `DISCORD_WEBHOOK` | Discord 웹훅 URL |
| `FE_ROLE_ID`      | FE 역할 ID       |

## 실행

```bash
python bot.py
```
